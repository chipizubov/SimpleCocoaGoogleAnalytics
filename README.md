SimpleCocoaGoogleAnalytics
==========================

A simple set of classes of using Google Analytics to track your OS X app's usage.

## How to use:

Add these to your app's project:
* AnalyticsHelper.h
* AnalyticsHelper.m
* AnalyticsEvent.h
* AnalyticsEvent.m

See the example app for how to begin reporting and send events.
